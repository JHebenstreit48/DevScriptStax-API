# Seeding & Local Dev

<hr class="dividerSection" />

## What Is Seeding?

<hr class="dividerSection" />

Seeding is the process of inserting starter or test data into a database so you can develop and test features without manually creating documents every time.

<hr class="dividerSection" />

## Running a Seed Script

<hr class="dividerSection" />

Run your project's seed script if defined in your project configuration:

```shell
npm run seed
```

<div class="centeredBullet">
  <ul class="diamondBullets fullWidthBullet">
    <li>Useful for local development and testing.</li>
    <li>Common after wiping or resetting your database.</li>
    <li>Depends on your project having a seed script set up.</li>
  </ul>
</div>
