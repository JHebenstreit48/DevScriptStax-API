# MongoDB Compass

<hr class="dividerSection" />

## What Is Compass?

<hr class="dividerSection" />

MongoDB Compass is a graphical interface for working with MongoDB databases.

<div class="centeredBullet">
  <ul class="diamondBullets fullWidthBullet">
    <li>View and edit <span class="emphasis">documents</span>.</li>
    <li>Run queries visually.</li>
    <li>Explore <span class="emphasis">collections</span> and <span class="emphasis">indexes</span>.</li>
    <li>Inspect schema structure.</li>
  </ul>
</div>

<hr class="dividerSection" />

## Connecting to a Database

<hr class="dividerSection" />

Compass connects to MongoDB using a <span class="emphasis">connection string</span>.

<div class="centeredBullet">
  <ul class="diamondBullets fullWidthBullet">
    <li>Can connect to a local MongoDB instance.</li>
    <li>Can connect to a MongoDB Atlas cluster.</li>
    <li>The connection string defines host, authentication, and database.</li>
  </ul>
</div>

<div class="xrefBox">
  <span class="emphasis">See:</span>
  <a href="/mongodb/tools/dev-workflow/connection-strings">
    MongoDB → Tools → Dev Workflow → Connection Strings & Environments
  </a>
</div>

<hr class="dividerSection" />

## Relationship to Other Tools

<hr class="dividerSection" />

<div class="centeredBullet">
  <ul class="diamondBullets fullWidthBullet">
    <li>Compass provides a visual alternative to shell-based interaction.</li>
    <li>It is commonly used alongside Atlas and local development environments.</li>
  </ul>
</div>
