# Functions and Modules

---
### Defining Functions
---

Functions are defined using the <span class="emphasis">def</span> keyword.

```python
def my_function():
    pass
```

---
### Function Parameters and Return Values
---

```python
def add(a, b):
    return a + b

result = add(2, 3)
```

---
### Modules
---

A module is a file containing Python code that can be reused in other files.

Example import:

```python
import math

python
from math import sqrt
```

---
### Built-in Functions
---

Some commonly used built-in functions:

```python
print()
round()
str()
float()
```

These functions are always available without importing anything.

---
### Summary
---

- Use <span class="emphasis">def</span> to define functions
- Functions can accept arguments and return values
- Modules help organize reusable code
- Python includes many built-in functions
