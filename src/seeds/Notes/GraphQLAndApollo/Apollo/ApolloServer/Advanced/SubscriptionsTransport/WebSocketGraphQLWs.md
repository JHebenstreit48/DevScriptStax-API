## Subscriptions Transport (WebSocket)

---

### Overview

Subscriptions enable real-time updates in GraphQL.

Instead of request/response like queries and mutations, subscriptions keep a connection open so the server can push updates when events occur.

---

### Real-Time Updates with Subscriptions

Common subscription use cases:

- Live notifications
- Chat messages
- Dashboard updates
- Collaboration events
