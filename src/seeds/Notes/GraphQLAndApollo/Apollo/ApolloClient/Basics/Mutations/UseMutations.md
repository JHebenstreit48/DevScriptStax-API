## useMutation

---

### Overview

useMutation is used to send GraphQL mutations from a client application.

Mutations are commonly used to change data on the server (create, update, delete actions).

---

### refetchQueries (Concept)

refetchQueries is an option that allows specified queries to be refetched after a mutation completes.

This is a common way to ensure UI data stays up-to-date after changes.
