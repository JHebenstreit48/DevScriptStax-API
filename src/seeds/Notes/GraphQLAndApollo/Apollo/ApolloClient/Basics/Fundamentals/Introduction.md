## Apollo Client Introduction

---

### Overview

Apollo Client is a client-side library for fetching and managing GraphQL data in an application.

It commonly provides:

- Query and mutation helpers
- Built-in caching
- Integration patterns for UI frameworks like React

---

### Apollo Client as State Management

Apollo Client often acts as a data layer that can:

- Fetch GraphQL data
- Cache results for fast reuse
- Keep UI in sync with server data
