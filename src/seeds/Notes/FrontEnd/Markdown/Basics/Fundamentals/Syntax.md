# Markdown Syntax Basics

This page covers the most fundamental building blocks of Markdown — everything you need to start writing structured content right away.

---

### Headers

Headers create hierarchy and are the easiest way to organize content.

```markdown
# H1 - Main Title (largest)
## H2 - Major Section
### H3 - Subsection
#### H4
##### H5
###### H6 - Smallest
```

---

### Text Formatting

Basic emphasis and strong text are extremely common.

```markdown
**Bold text** or __bold text__
*Italic text* or _italic text_
**Bold and *nested italic* text**
~~Strikethrough~~
```

---

### Lists

#### Unordered Lists

```markdown
- Item one
- Item two
  - Nested item
- Item three
```

#### Ordered Lists

```markdown
1. First step
2. Second step
   1. Sub-step
3. Third step
```

---

### Links and Images (Basic)

Links and images use very similar syntax.

```markdown
[Link text](https://example.com)

![Alt text](https://example.com/image.jpg)
```

---
### Block Elements
---

Horizontal rules and blockquotes help separate content visually.

```markdown
> This is a blockquote.
> It can span multiple lines.
```

```markdown
Horizontal rules (---) create visual breaks.
```

---

These are the core syntax elements used in almost every Markdown document. More advanced features (tables, code blocks, task lists, etc.) are covered in later sections.
