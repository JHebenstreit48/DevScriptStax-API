# Introduction to Markdown

Markdown is a <span class="emphasis">lightweight markup language</span> designed to be easy to read and write. It allows users to create formatted text using a plain-text editor that can later convert into HTML.

The primary purpose of Markdown is to simplify the process of creating web-friendly text with <span class="emphasis">straightforward, readable syntax</span>.
