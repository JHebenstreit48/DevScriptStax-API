# What Is HTML?

<hr class="dividerSection" />

<span class="emphasis">Hypertext Markup Language</span> <span class="secondEmphasis">(HTML)</span> is one of the cornerstone <span class="emphasis">languages</span> of the web.  
It is the <span class="emphasis">standard</span> markup language for creating <span class="emphasis">documents</span> to be displayed in a <span class="emphasis">web browser</span>.

<hr class="dividerSection" />

## Structure of HTML

<hr class="dividerSection" />

<span class="emphasis">HTML</span> defines the <span class="emphasis">structure</span> of a webpage, similar to how a skeleton provides support for the human body.  
HTML is composed of <span class="emphasis">elements</span>, which define the <span class="emphasis">content</span> and framework of a webpage.

## DOCTYPE Declaration

<hr class="dividerSection" />

The <span class="emphasis">&lt;!DOCTYPE&gt;</span> declaration defines the document type and version of HTML being used.  
It must be the first line of an HTML document for proper browser rendering.

```html
<!DOCTYPE html>
```

<hr class="dividerSection" />

## HTML Element

<hr class="dividerSection" />

The <span class="codeSnip">&lt;html&gt;</span> element serves as the root of an HTML document.

All other elements must be contained within this tag.

```html
<html>
  ...
</html>
```

<hr class="dividerSection" />

## Viewing Page Source

<hr class="dividerSection" />

Right-clicking anywhere on a webpage in the browser and selecting <span class="emphasis">View Page Source</span> allows you to see the raw HTML that makes up that page.

This works on any webpage and is a useful way to inspect how a page is structured.
