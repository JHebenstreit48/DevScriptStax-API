# Semantic Tags

<hr class="dividerSection" />

<hr class="dividerSubsection1" />

## Header Tag

<hr class="dividerSubsection1" />

The <span class="secondEmphasis">&lt;</span><span class="emphasis">header</span><span class="secondEmphasis">&gt;</span> element represents introductory content or navigational links.

```html
<header>
  <h1>Page Header</h1>
</header>
```

<hr class="dividerSubsection1" />

## Article Tag

<hr class="dividerSubsection1" />

The <span class="secondEmphasis">&lt;</span><span class="emphasis">article</span><span class="secondEmphasis">&gt;</span> element represents a self-contained piece of content.

```html
<article>
  <h2>Article Title</h2>
  <p>Article content goes here.</p>
</article>
```

<hr class="dividerSubsection1" />

## Footer Tag

<hr class="dividerSubsection1" />

The <span class="secondEmphasis">&lt;</span><span class="emphasis">footer</span><span class="secondEmphasis">&gt;</span> element represents footer content, often containing copyright or contact information.

```html
<footer>
  <p>Footer content here.</p>
</footer>
```



<hr class="dividerSubsection1" />

## Nav Tag

<hr class="dividerSubsection1" />

The <span class="secondEmphasis">&lt;</span><span class="emphasis">nav</span><span class="secondEmphasis">&gt;</span> element defines a set of navigation links, typically used for menus or page navigation.

```html
<nav>
  <a href="/home">Home</a>
  <a href="/about">About</a>
  <a href="/contact">Contact</a>
</nav>
```

<hr class="dividerSubsection1" />

## Main Tag

<hr class="dividerSubsection1" />

The <span class="secondEmphasis">&lt;</span><span class="emphasis">main</span><span class="secondEmphasis">&gt;</span> element represents the primary content of the document. It should only be used once per page and excludes repeated content like headers, footers, and navigation.

```html
<main>
  <h1>Welcome to the Page</h1>
  <p>This is the main content area.</p>
</main>
```

<hr class="dividerSubsection1" />

## Section Tag

<hr class="dividerSubsection1" />

The <span class="secondEmphasis">&lt;</span><span class="emphasis">section</span><span class="secondEmphasis">&gt;</span> element represents a standalone section of related content within a document. Unlike <span class="secondEmphasis">&lt;</span><span class="emphasis">article</span><span class="secondEmphasis">&gt;</span>, a section is not necessarily self-contained and is typically part of a larger whole.

```html
<section>
  <h2>Section Title</h2>
  <p>This is a grouped section of related content.</p>
</section>
```

<hr class="dividerSubsection1" />

## Aside Tag

<hr class="dividerSubsection1" />

The <span class="secondEmphasis">&lt;</span><span class="emphasis">aside</span><span class="secondEmphasis">&gt;</span> element represents content that is tangentially related to the surrounding content, such as a sidebar, pull quote, or supplementary information.

```html
<aside>
  <h3>Related Links</h3>
  <p>Additional information or sidebar content goes here.</p>
</aside>
```
