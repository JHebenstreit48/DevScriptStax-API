# Individual Tags

<hr class="dividerSection" />

<hr class="dividerSubsection1" />

## Form Tag

<hr class="dividerSubsection1" />

The <span class="secondEmphasis">&lt;</span><span class="emphasis">form</span><span class="secondEmphasis">&gt;</span> element creates a form for collecting user input. The action attribute specifies where to send the form data.

```html
<form>...</form>
```

<hr class="dividerSubsection1" />

## Label Tag

<hr class="dividerSubsection1" />

The <span class="secondEmphasis">&lt;</span><span class="emphasis">label</span><span class="secondEmphasis">&gt;</span> element provides a description for form inputs, improving accessibility.

```html
<label for="name">Name:</label>
```

<hr class="dividerSubsection1" />

## Input Tag

<hr class="dividerSubsection1" />

The <span class="secondEmphasis">&lt;</span><span class="emphasis">input</span><span class="secondEmphasis">&gt;</span> element creates various types of form fields, such as text fields, email inputs, and checkboxes.

```html
<input type="text" id="name" name="name" />
```

<hr class="dividerSubsection1" />

## Textarea Tag

<hr class="dividerSubsection1" />

The <span class="secondEmphasis">&lt;</span><span class="emphasis">textarea</span><span class="secondEmphasis">&gt;</span> element allows users to input multi-line text.

```html
<textarea rows="4" cols="50">Enter your text here...</textarea>
```

<hr class="dividerSubsection1" />

## Button Tag

<hr class="dividerSubsection1" />

The <span class="secondEmphasis">&lt;</span><span class="emphasis">button</span><span class="secondEmphasis">&gt;</span> element creates a clickable button, often used to submit forms.

```html
<button type="submit">Submit</button>
```
